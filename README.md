### 功能列表
+ 推荐、最热、待解决问答首页
+ 问答详情页面，包含关注问题、点赞问题、点赞评论、回复评论、展开更多功能
+ 语音回答、文字回答功能
+ 搜索功能
+ 发布问题功能
+ 关注、点赞、评论、提交等模板消息的搜集
+ 个人中心（我关注的问题、我的提问、我的消息、我的回答、关于）
